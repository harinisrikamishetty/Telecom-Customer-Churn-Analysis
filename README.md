# 📊 Telecom Customer Churn Analysis

## 📌 Project Overview

This project focuses on analyzing customer churn in a telecom company using SQL, Python, and Power BI. The main goal of the project is to identify the factors that influence customer churn and understand customer behavior through data analysis and visualization.

Customer churn refers to customers who stop using the company’s services. By analyzing churn patterns, the company can improve customer retention strategies and reduce customer loss.

# 🔍 Project Workflow

## 1. Data Collection

The telecom customer dataset was collected in CSV format containing customer demographics, services, billing information, and churn details.

## 2. Data Cleaning

Data cleaning was performed using Python and Pandas. Missing values, incorrect data types, and unnecessary columns were handled to prepare the dataset for analysis.

## 3. SQL Analysis

SQL queries were used to analyze:

* Total customers
* Churn customers
* Churn percentage
* Contract type analysis
* Payment method analysis
* Internet service analysis
* Tech support and online security analysis

## 4. Data Visualization

Power BI was used to create interactive dashboards and visualizations such as:

* KPI Cards
* Donut Charts
* Bar Charts
* Tree Maps
* Pie Charts
* Slicers and Filters

---

# 📈 Key Insights

* The company has a churn rate of 26.5%.
* Customers with Month-to-Month contracts have the highest churn.
* Customers using Electronic Check payment methods churn more.
* Fiber Optic users have higher churn compared to DSL users.
* Customers with higher monthly charges are more likely to leave.
* Customers without Tech Support and Online Security show higher churn rates.
* Long-term contract customers are more likely to stay with the company.
